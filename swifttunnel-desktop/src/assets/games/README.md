# Game library art

Drop a **vertical poster** image here named after the game `id` and it loads
automatically in the Games tab — no code change needed.

- File name: `<id>.<ext>` where `<ext>` is `png`, `jpg`, `jpeg`, or `webp`
- Current ids: `roblox`
- Recommended: portrait artwork, ~3:4 ratio (e.g. 600×800), object-cover fit

Examples:

```
roblox.jpg
fortnite.png
genshin.webp
```

If no file matches a game id, the card shows a branded poster fallback.
