## What's changed

- Redesigned the entire app around a new shell: a fixed, collapsible sidebar with grouped navigation and a live connection card, plus a header bar that always shows your tunnel status and ping.
- Rebuilt the Connect screen as a command deck: an animated plasma status orb, larger region display, and an always-visible Latency / Session / Routing stats strip.
- Refreshed the region picker with latency signal bars, relay counts, and pinned-server info on every row.
- Restyled the Settings page with a new account card, cleaner update flow, and tidier tunnel/adapter controls.
- Moved the SwiftTunnel Power Plan toggle from the Roblox boost page into Optimize → System, where it now applies instantly.
- The app now opens at a larger default window size that fits the new layout.
- Added native Windows ARM64 support — Snapdragon/ARM PCs now get their own installer and updates instead of running through emulation.
- IPv6 leak blocking now runs through the WinpkFilter driver, giving stronger protection against IPv6 leaks while tunneled.
- The firewall repair tool now keeps a registry backup and can restore from it, so firewall fixes are safer to apply and revert.
