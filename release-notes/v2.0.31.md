## What's changed

- UltraBoost and FPS unlock now apply correctly for Bloxstrap, Fishstrap, Bubblestrap, Froststrap, and Voidstrap installs.
- SwiftTunnel writes the Roblox FPS unlock FFlag in the expected integer format for supported bootstrapper launchers.
- Boost cleanup now removes only SwiftTunnel-owned FFlags, preserves user-owned FastFlags, and leaves malformed ClientAppSettings files untouched instead of overwriting them.
- Empty bootstrapper ClientSettings folders created by SwiftTunnel are cleaned up when Boost/FPS settings are disabled or uninstalled.

**Release**: https://github.com/Swift-tunnel/swifttunnel-app/releases/tag/v2.0.31
