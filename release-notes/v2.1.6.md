## What's changed

- Reduces WebView2 and Tauri startup/resource overhead while preserving the same SwiftTunnel desktop behavior.
- Removes the QoS boost from PC Boosts so SwiftTunnel no longer changes Windows QoS registry policy.
- Tightens PC Boosts cleanup so stale Roblox GPU preferences and boost registry snapshots are restored more reliably.
