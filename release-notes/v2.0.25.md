## What's changed

- Fixed legacy network boost settings so older installs that only saved the master boost flag now restore the intended Nagle, Windows throttling, and gaming QoS boosts.
- Reapply saved network boost settings on Windows startup, so boosts are active before the Boost tab is opened.
- Kept the master boost flag synchronized with the real per-boost settings to prevent stale all-off configs from turning boosts back on.
