## What's changed

- Fixed a "Connected but no traffic" failure where Roblox launches could lose tunnel coverage if the first owner-table refresh ran before any sockets appeared. Recently-detected processes now survive an empty refresh long enough to take effect.
- Repeated network-unreachable relay send failures now surface as a real connection error instead of silently sitting Connected with no game traffic. The app will report the problem and clean up so you can pick another server.
- Persistent low-level send errors (permissions, buffer exhaustion) now escalate the same way, so the connection no longer pretends to be healthy when every send is failing.
