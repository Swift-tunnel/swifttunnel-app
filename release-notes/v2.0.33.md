## What's changed

- Fixed Roblox critical settings launch failures for users whose local DNS is blocked or country-shaped.
- When API tunneling is enabled, SwiftTunnel now repairs only the exact Roblox critical settings hostnames with temporary DNS-over-HTTPS hosts entries.
- The repair is cleaned up automatically on disconnect, startup recovery, and crash-safe drop paths.
