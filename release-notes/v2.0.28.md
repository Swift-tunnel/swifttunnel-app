## What's changed

- Added the SwiftTunnel Power Plan boost, a dedicated Windows power profile for lower-latency gaming sessions.
- Made the Performance boost preset enable the SwiftTunnel power plan automatically while still restoring your previous plan when boosts are disabled.
- Hardened power-plan recovery so SwiftTunnel imports the plan only when needed, cleans up staged files, and avoids deleting the active plan during fallback recovery.

**Release**: https://github.com/Swift-tunnel/swifttunnel-app/releases/tag/v2.0.28
