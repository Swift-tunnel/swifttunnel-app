## What's changed

- Fixed false "relay dead" disconnects while sitting in idle game states such as lobbies or menus. SwiftTunnel now treats valid relay PONG responses as liveness, so responsive relays no longer get killed just because no game packets are flowing.
- Reworked relay keepalives to use the supported PING/PONG protocol instead of session-only packets that the relay never answers.
- Reset relay health when switching relays, so a new relay does not inherit stale silence from the previous one.
