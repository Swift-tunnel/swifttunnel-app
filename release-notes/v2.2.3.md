## What's changed

- Fixed Auto Route only switching once per session: leaving a game or teleporting now re-detects the new game server's region and switches your relay again. Joins that happen while another game connection is still active are left on the current relay on purpose — switching mid-game would disconnect you from the server.
- Fixed the Connect tab showing the previous server's ping after Auto Route switched relays. The reading now resets on every switch and falls back automatically when the relay stops responding.
- Auto Route switches now fully authenticate with the new relay before moving any traffic. If authentication fails, you stay connected through your current working server instead of losing traffic or dropping protection.
- Fixed the server ping test on non-English versions of Windows (German, French, Spanish, and others), which silently broke latency measurements and Auto Route's server scoring.
- Auto Route, region whitelist, and pinned-server changes now apply immediately when you save settings — no reconnect needed. Turning Auto Route off mid-session now also releases any in-flight route changes right away, and turning it on mid-game will never re-route a match you are already playing.
- Smoother app performance during the periodic server ping test.
