## What's changed

- Added startup detection and repair for broken split-tunnel driver states. SwiftTunnel now checks the driver and selected adapter during the loading screen, repairs only when needed, and notifies the user to restart Windows if the repair requires it.

- Fixed Split tunnel driver repair saying "Fixed" when the selected or active adapter was still missing the WinpkFilter binding. Repair now validates the same adapter path Connect uses.

- Improved recovery for the "no TCP/IP-bound network adapters were enumerated" driver state so startup repair and manual repair can still run even when adapter discovery is already broken.

- Improved Vietnam / Route Assist Roblox loading reliability by keeping direct Roblox menu, chat, avatar, settings, and asset hosts on normal DNS instead of stale hosts-file pins, while still relaying the Roblox join/control path.

**Release**: https://github.com/Swift-tunnel/swifttunnel-app/releases/tag/v2.5.11
