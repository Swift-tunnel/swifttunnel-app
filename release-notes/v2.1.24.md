## What's changed

**New**
- **Optimization tab** — one-click, fully reversible Windows tweaks for FPS and latency: mouse acceleration, visual effects, transparency, background apps, Game Bar/DVR, power throttling, core parking, SysMain, GPU scheduling, and telemetry. Each shows exactly what it changes and can be toggled back off. The system memory cleaner now lives here too.
- **Games library** — pick a game to optimize (Roblox-only for now).
- **Bypass Country Bans is back** — restored in the Roblox optimize section, running independently of Route Assist (neither can break the other).

**Fixes**
- Fixed Roblox **"Failed to download or apply critical settings"** by keeping Roblox's critical bootstrap hosts direct.
- Fixed slow/failed Roblox connections by pinning multiple reachable Roblox edge IPs during DNS repair (also fixes bootstrappers like Bloxstrap/Fishstrap timing out on `clientsettings`).
- Fixed the app sometimes needing **two launches to open** — it now appears immediately.
- Fixed the app failing to start on **older Windows 10** builds (1809/1903/1909).
