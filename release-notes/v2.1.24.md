## What's changed

- Fixed Roblox bootstrappers (Bloxstrap, Fishstrap, Frostrap) failing to connect to `clientsettings.roblox.com` while SwiftTunnel was active. The DNS repair now pins multiple Roblox edge IPs and only uses ones it can actually reach, so a single unreachable edge no longer blocks Roblox updates.

**Release**: https://github.com/Swift-tunnel/swifttunnel-app/releases/tag/v2.1.24
