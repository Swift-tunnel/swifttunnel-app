## What's changed

- Added a confirmation dialog before the destructive desktop uninstall flow runs, with explicit cleanup wording and a recoverable success/failure UI state.
- Fixed the Connect primary action copy while disconnecting so the hero and button both show the disconnecting transition instead of mixing in "Connecting...".
- Made the server picker keyboard-reachable without invalid nested buttons, and added accessible names to desktop switches so screen reader and voice users can identify each setting.
- Preserved unapplied Boost changes when Roblox restart fails, reduced noisy saved-toasts from continuous settings inputs, and wired updater download progress events into the frontend progress bar.

**Release**: https://github.com/Swift-tunnel/swifttunnel-app/releases/tag/v2.0.22
