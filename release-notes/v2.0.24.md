## What's changed

- Fixed auto-routing choosing the wrong region when Roblox contacts API or infrastructure endpoints during play.
- Auto-routing now only switches relays for real Roblox game-server traffic with a structured resolver result.
- Prevented mid-game relay swaps from later Roblox-owned endpoints once the current game server has been identified.
