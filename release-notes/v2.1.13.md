## What's changed

- Restored the Bypass country bans control for Roblox country-level DPI blocks.
- GoodbyeDPI now tries modes -1 through -9 automatically and keeps the first mode that can reach Roblox.
- Kept Route Assist separate, so relay routing still works even if the local DPI bypass is unavailable on a network.
- Added an in-app warning when the country-ban bypass cannot be used on the current network.
