## What's changed

- Fixed connect failures on PCs where IPv6 is already disabled on the active network adapter.
- SwiftTunnel now skips the IPv6 leak-prevention firewall rule when there is no IPv6 path to block, while still blocking IPv6 on adapters that have it enabled.
