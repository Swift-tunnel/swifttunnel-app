## What's changed

- Fixed Partial Bypass for Roblox teleports and server-hops.

- Improved Partial Bypass asset loading so maps, textures, sounds, avatars, and accessories can use the relay while gameplay UDP stays direct for normal ping.

- Kept Full Country Ban as the full relay option for countries where Roblox website, app, and game traffic are fully blocked.

- Added an in-app sign-in fallback for users whose browser login opens but does not complete inside SwiftTunnel.

- Improved PPPoE and recreated-adapter handling so Manual adapter mode can fall back safely instead of blocking the connection.

- Made split-tunnel driver recovery safer after BSOD reports by avoiding automatic kernel driver repair loops when Windows already has the driver installed but unstable.

- Improved WinpkFilter binding handling so common split-tunnel driver errors give restart/contact-support guidance instead of telling users to spam Repair.

- Reduced random PowerShell/MSI repair popups during connect and startup.

- Fixed missing SwiftTunnel logos after repair or restart flows.

- Updated internal tests around driver repair, adapter fallback, login fallback, and Partial Bypass routing.
