## What's changed

- Improved Repair Center error handling so failed or incomplete checks keep the original driver, adapter, startup, and tunnel context for support.
- Made Tunnel cleanup safer by clearing only stale tunnel state instead of running the broader system cleanup path.
- Added stronger Repair Center verification so cleanup reports partial when VPN state, diagnostics, or driver health still need attention.
