## What's changed

**Fixes**
- Added a Windows Firewall repair in Repair Center for PCs where SwiftTunnel cannot install its temporary IPv6 leak-prevention rule and connect fails with an `advfirewall` command error.
- The repair checks the Windows Firewall services, waits for services that are still starting, backs up the current firewall policy, resets Windows Firewall only after the backup succeeds, and verifies that `advfirewall` commands work again before reporting success.
- Repair results now include support-ready details for admin state, service state, firewall backup path, and before/after firewall checks.

**Release**: https://github.com/Swift-tunnel/swifttunnel-app/releases/tag/v2.1.27
