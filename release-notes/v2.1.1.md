## What's changed

- Refined monochrome UI redesign: new Geist + Geist Mono typography, three-tier surface palette, and a consistent hero-card layout across the Connect, Boost, Diagnostics, and Settings tabs.
- LCD-style numeric readouts and pill status indicators give latency and connection state a cleaner, more tactile look.
- Region rows now use fixed-width slots so 2- and 3-digit pings stay aligned across every row, regardless of whether the region has multiple servers.
- Sidebar drops the keyboard-shortcut column and refines the user tile; primary buttons gain a gradient gloss and the connect toggle gets inset shadows for a more modern feel.
