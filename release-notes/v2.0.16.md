## What's changed

- Auto Routing now trusts the IPinfo-backed SwiftTunnel resolver for game server regions and no longer guesses from the old hardcoded Roblox IP table when lookups fail.
- If the resolver cannot return a structured region, SwiftTunnel keeps routing on the current relay instead of switching to a guessed region.
- Auto Routing relay switches now show a Windows toast that says which region you were routed to, using the new Swifty notification icon.

**Release**: https://github.com/Swift-tunnel/swifttunnel-app/releases/tag/v2.0.16
