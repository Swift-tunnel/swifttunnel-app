## What's changed

- Added Roblox Route Assist, a clearer replacement for API Tunneling that can route Roblox login and API traffic through your selected tunnel region.
- Route Assist now handles browser-owned Roblox HTTP(S) while keeping unrelated browser traffic off the tunnel.
- The setting is now visible on the Connect page and in Settings with clearer guidance for network-ban and region-placement use cases.

**Release**: https://github.com/Swift-tunnel/swifttunnel-app/releases/tag/v2.0.35
