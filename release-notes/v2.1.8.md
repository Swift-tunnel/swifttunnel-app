## What's changed

- Restored low-jitter relay traffic marking for users affected by ping spikes after updating from v2.1.5.
- SwiftTunnel now marks only its own V3 relay UDP traffic with DSCP EF (46) and keeps the old broad Roblox/Microsoft Store Gaming QoS policies removed.
- Hardened Windows QoS cleanup so failed policy installs roll back registry changes instead of leaving partial system QoS state behind.

**Release**: https://github.com/Swift-tunnel/swifttunnel-app/releases/tag/v2.1.8
