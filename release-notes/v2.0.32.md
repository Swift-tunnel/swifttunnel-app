## What's changed

- Fixed Roblox critical settings fetches failing when API tunneling is enabled.
- DNS traffic now stays on the local connection, so Roblox can resolve launch-time settings domains before game/API traffic is routed through SwiftTunnel.
- API tunneling still routes Roblox HTTPS and game traffic through the selected relay; only DNS is bypassed.

**Release**: https://github.com/Swift-tunnel/swifttunnel-app/releases/tag/v2.0.32
