## What's changed

- Hardened VPN reconnect and relay selection paths so stale relay send failures, missing relay candidates, and unavailable forced servers are handled explicitly instead of silently falling back.
- Improved tunnel safety by blocking loopback, link-local, multicast, broadcast, and other non-tunnelable destinations before packets are sent to the relay.
- Tightened custom relay and geolocation behavior: custom relays stay tester-only and still run ticket policy checks, while low-confidence Roblox region lookups no longer guess a fallback server.
- Fixed recovery edge cases around IPv6 firewall restore verification, WFP unblock logging, process-monitor cleanup, and Roblox restart handling so the generic Windows Store host is never force-killed.
