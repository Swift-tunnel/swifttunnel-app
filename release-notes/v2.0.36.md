## What's changed

- Fixed Roblox Route Assist missing some launch/API traffic when Roblox critical-settings hosts resolve to CDN IPs.
- Route Assist now tunnels the exact temporary CDN IPs resolved for repaired Roblox bootstrap hostnames, while unrelated browser traffic still stays outside the tunnel.
- Disconnect and cleanup now clear the temporary bootstrap route IP set so stale repair state cannot affect later sessions.
