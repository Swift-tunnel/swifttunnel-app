## What's changed

- Added hardware-ban enforcement for desktop logins so a banned account can flag that Windows device for future SwiftTunnel accounts on the same machine.
- Kept the device identifier private to SwiftTunnel-owned API calls and out of browser OAuth URLs.
- Added safeguards for missing or placeholder Windows machine IDs so unrelated devices are not grouped together by mistake.
