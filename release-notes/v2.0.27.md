## What's changed

- Added admin ban enforcement in the desktop app. Banned accounts now see a blocked-account screen, and SwiftTunnel disconnects VPN plus restores active boosts when a ban is detected.
- Made Roblox FastFlags survive Roblox version updates by applying saved flags across all valid `version-*` folders instead of only the previous active folder.
- Improved ban handling during login, startup, profile refresh, and VPN connect so structured ban responses are surfaced consistently without false positives from unrelated error text.
