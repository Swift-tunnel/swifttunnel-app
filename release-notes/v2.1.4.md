# SwiftTunnel v2.1.4

This hotfix keeps Windows builds compatible with older Windows environments after the v2.1.3 driver-selection update.

- Fixes a startup crash in VPN helper binaries on Windows builds that do not export the optional `GetMachineTypeAttributes` API.
- Keeps ARM64 driver detection intact on supported Windows hosts while safely falling back on older systems.

**Release**: https://github.com/Swift-tunnel/swifttunnel-app/releases/tag/v2.1.4
