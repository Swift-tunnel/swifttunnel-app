## What's changed

Changed normal SwiftTunnel tunneling to focus on Roblox gameplay traffic only, so regular Connect improves game ping without dragging textures, avatars, menus, chat, or loading assets through the relay.

Fixed Roblox UDP/443 asset/API traffic being treated like gameplay traffic.

Fixed stale routing cache behavior.

Improved Partial Bypass behavior.

Kept Full Country Ban.

Fixed Full Country Ban browser routing for fully blocked countries: Roblox website traffic can now use the relay even before DNS/host pins are ready, while Route Assist and Partial Bypass still keep unpinned browser traffic direct.

Improved PPPoE/WAN adapter selection.

Fixed Manual adapter mode hard-failing on PPPoE or recreated adapters. If the saved manual adapter is unavailable, SwiftTunnel now falls back to Smart Auto instead of blocking the connection.

Improved Roblox overlay startup behavior so FPS/metrics can recover when the overlay is enabled after Roblox is already running.

Release: https://github.com/Swift-tunnel/swifttunnel-app/releases/tag/v2.5.13
