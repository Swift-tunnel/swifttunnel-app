## What's changed

- Added a loading preparation screen so SwiftTunnel opens visibly while it checks for leftover network state from crashes or failed driver sessions.

- Improved startup self-repair for stale SwiftTunnel network state, including leftover WinpkFilter bindings, tunnel-mode markers, IPv6 recovery markers, and adapter offload recovery.

- Added background connectivity maintenance on launch: SwiftTunnel now refreshes DNS and asks Windows Time to resync, which can help Roblox login/authentication issues caused by stale DNS or a bad system clock.

- Improved split-tunnel repair guidance.

**Release**: https://github.com/Swift-tunnel/swifttunnel-app/releases/tag/v2.5.9
