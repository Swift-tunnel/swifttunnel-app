## What's changed

- Fixed Roblox Studio failing while SwiftTunnel is connected. Studio now stays direct unless Full Country Ban bypass is enabled.

- Improved Bypass Country Ban reliability for fully blocked networks. SwiftTunnel now enters full relay routing before Roblox DNS repair, so stale direct-path pins cannot pull Roblox settings or assets back onto a blocked connection when secure DNS is blocked.

- GoodbyeDPI is now treated as a best-effort helper instead of blocking the connection. It checks Roblox over HTTPS and falls back to the relay quickly if no local DPI mode works.

- Added Internet Recovery repair for the "No Internet, Secured" issue after a crash by unbinding leftover WinpkFilter / nt_ndisrd adapter bindings from Repair.

- Fixed Auto Route not re-routing cleanly after a Roblox match/server goes quiet, so new matches can trigger fresh region routing without disconnecting SwiftTunnel.

- Improved login/update HTTPS compatibility on PCs with antivirus or proxy TLS inspection by trusting Windows native certificate roots.
